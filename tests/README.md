# Tests Directory

This directory contains all test files for the TSV Ledger application.

## Test Files

- `test-amazon-edit-feature.js` - Tests Amazon item editing functionality
- `test-ux-amazon-edit.js` - Tests user experience for Amazon editing
- `test-master.js` - Master test orchestration
- `test-complete.js` - Complete test suite runner

## Running Tests

From the project root:
```bash
node tests/test-complete.js
```
