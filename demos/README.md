# Demos Directory

This directory contains demonstration scripts showcasing various features of the TSV Ledger application.

## Files

- `ai-demo.js` - AI analysis feature demonstration
- `demo-amazon-edit.js` - Amazon item editing functionality demo
- `employee-benefits-demo.js` - Employee benefits analysis demo
- `premium-demo.js` - Premium features demonstration

## Usage

Run demos individually:
```bash
node demos/ai-demo.js
node demos/demo-amazon-edit.js
```
