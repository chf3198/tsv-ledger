# Servers Directory

This directory contains alternative server implementations and backup files.

## Files

- `basic-server.js` - Minimal basic server implementation
- `minimal-server.js` - Stripped-down server for testing
- `server-minimal.js` - Another minimal server variant

## Usage

Run alternative servers:
```bash
node servers/basic-server.js
node servers/minimal-server.js
```

Note: The main server is `server.js` in the root directory.
