TSV Categorizer - Business Intelligence Engine Advanced categorization and analysis engine for Texas Sunset Venues expense tracking. Provides intelligent categorization of expenses using keyword matching and business rules, along with comprehensive analytics for spending patterns and business insights.

categorize(expense) → {string}

Categorize a single expense based on its description using keyword matching This method performs case-insensitive keyword matching against the expense description to determine the most appropriate business category. It uses a first-match approach with predefined business rules for Texas Sunset Venues operations.
