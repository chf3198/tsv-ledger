## Interactive Tutorials

### Tutorial 1: Your First Data Import

#### Objective
Learn how to import expense data and see immediate results.

#### Prerequisites
- Basic understanding of CSV files
- Access to expense data (bank statements, credit card reports)

#### Step-by-Step Guide

##### Step 1: Access the Data Import Page
1. Open TSV Ledger in your browser
2. Navigate to the "Data Import" section from the main menu
3. You'll see the import interface with multiple options

##### Step 2: Prepare Your Data
**Supported Formats:**
- CSV files from banks/financial institutions
- Amazon order history exports
- Manual expense entry

**Data Requirements:**
```
Required columns: date, description, amount
Optional columns: category, merchant, notes
Date format: MM/DD/YYYY or YYYY-MM-DD
Amount format: -123.45 (negative for expenses)
```

##### Step 3: Upload Your File
1. Click "Choose File" button
2. Select your CSV file
3. Click "Upload and Process"
4. Wait for the processing animation to complete

##### Step 4: Review Import Results
The system will show:
- ✅ Successfully imported: X transactions
- ⚠️ Warnings: Y items need review
- ❌ Errors: Z items couldn't be processed

##### Step 5: Categorize Uncertain Transactions
For transactions that couldn't be automatically categorized:
1. Click on the transaction row
2. Select the appropriate category from the dropdown
3. Add notes if needed
4. Click "Save Changes"

##### Step 6: View Your Dashboard
Navigate to the main dashboard to see:
- Total expenses by category
- Monthly spending trends
- Recent transactions
- AI-generated insights

#### Practice Exercise
1. Export a month's worth of transactions from your bank
2. Import the data into TSV Ledger
3. Manually categorize at least 5 uncertain transactions
4. Generate a spending report

#### Expected Outcomes
- Successfully import expense data
- Understand the categorization process
- View basic analytics on the dashboard
- Generate simple reports

### Tutorial 2: Amazon Order Integration

#### Objective
Import and analyze Amazon purchase history for comprehensive expense tracking.

#### Prerequisites
- Amazon account with order history
- Understanding of ZIP file downloads

#### Step-by-Step Guide

##### Step 1: Download Amazon Data
1. Log into your Amazon account
2. Go to "Your Account" → "Order History"
3. Click "Download order reports"
4. Select date range (recommend last 6-12 months)
5. Choose "CSV" format
6. Download the ZIP file

##### Step 2: Access Amazon Import
1. In TSV Ledger, navigate to "Amazon Import"
2. Click "Choose ZIP File"
3. Select your downloaded Amazon ZIP file

##### Step 3: Process Amazon Data
The system will automatically:
- Extract files from the ZIP archive
- Parse order data, cart history, and returns
- Categorize Amazon purchases
- Calculate shipping and tax amounts
- Identify subscription services

##### Step 4: Review Processing Results
Check the processing summary:
- Orders processed: X
- Items categorized: Y
- Subscription services detected: Z
- Returns processed: W

##### Step 5: Analyze Amazon Spending
View Amazon-specific insights:
- Monthly Amazon spending trends
- Category breakdown (Electronics, Books, etc.)
- Subscription service costs
- Prime membership value analysis

##### Step 6: Integrate with Overall Budget
Amazon data is now part of your complete expense picture:
- Combined spending across all sources
- Cross-category analysis
- Subscription cost optimization

#### Practice Exercise
1. Download your Amazon order history
2. Import the ZIP file into TSV Ledger
3. Review subscription services identified
4. Compare Amazon spending to total budget

#### Expected Outcomes
- Seamless Amazon data integration
- Automatic subscription detection
- Enhanced expense categorization
- Comprehensive spending analysis

### Tutorial 3: AI-Powered Analysis

#### Objective
Leverage AI analysis for spending insights and anomaly detection.

#### Prerequisites
- Imported expense data (minimum 3 months recommended)
- Basic understanding of expense categories

#### Step-by-Step Guide

##### Step 1: Access AI Analysis
1. Navigate to "AI Insights" section
2. View the analysis dashboard
3. Review current AI processing status

##### Step 2: Understand AI Categories
The AI system automatically categorizes expenses into:
- **Food & Dining**: Restaurants, groceries, delivery
- **Transportation**: Gas, public transit, rideshare
- **Shopping**: Retail purchases, online shopping
- **Entertainment**: Movies, games, subscriptions
- **Bills & Utilities**: Rent, electricity, internet
- **Healthcare**: Medical expenses, pharmacy
- **Travel**: Flights, hotels, vacation spending
- **Other**: Miscellaneous expenses

##### Step 3: Review AI Insights
Explore different analysis views:

**Spending Patterns:**
- Weekly spending cycles
- Seasonal spending trends
- Category growth/decline

**Anomaly Detection:**
- Unusual spending spikes
- Out-of-pattern purchases
- Potential fraudulent activity

**Recommendations:**
- Budget optimization suggestions
- Subscription cost analysis
- Spending reduction opportunities

##### Step 4: Interactive Analysis
Use the interactive features:
1. Click on any data point for details
2. Filter by date range or category
3. Export insights to PDF/CSV
4. Set up automated alerts

##### Step 5: Apply AI Recommendations
For each recommendation:
1. Review the suggestion details
2. Click "Apply" to implement changes
3. Or "Dismiss" with feedback
4. Track recommendation impact over time

#### Practice Exercise
1. Review AI-generated spending insights
2. Identify 3 spending anomalies
3. Apply 2 budget optimization recommendations
4. Set up an automated spending alert

#### Expected Outcomes
- Understanding of AI analysis capabilities
- Identification of spending patterns
- Implementation of optimization recommendations
- Proactive budget management
