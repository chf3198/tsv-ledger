## Customization Guidelines

### Technology Stack Adaptation
- **JavaScript/Node.js**: Focus on npm scripts, module structure
- **Python**: Emphasize package structure, virtual environments  
- **Java**: Standard Maven/Gradle patterns, package organization
- **C#/.NET**: Solution structure, namespace organization
- **Go**: Module structure, package organization

### Team Size Considerations
- **Solo Developer**: Simplified patterns, focus on future handoff
- **Small Team (2-5)**: Lightweight processes, direct communication
- **Medium Team (6-15)**: Formal processes, clear responsibilities
- **Large Team (15+)**: Standardized patterns, automated enforcement

### Project Type Adaptations
- **Web Applications**: Frontend/backend separation, asset organization
- **APIs/Services**: Service layer organization, documentation focus
- **Libraries/Frameworks**: Public interface design, example organization
- **Mobile Apps**: Platform-specific considerations, resource organization
